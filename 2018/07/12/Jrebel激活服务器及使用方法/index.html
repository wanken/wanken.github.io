激活步骤:这里以IDEA为例

  1. 菜单栏 Intellij IDEA->Preference…
  2. 选择 JRebel->JRebel License
  3. 点击 Connect to License Server
  4. 填入服务器 http://65.49.200.145:8081/{username} {username}替换为任何guid即可(末尾没有斜线”/“)
  5. 填入邮箱 邮箱随便填(jokerwaver@gmail.com)
  6. 点击 Activate
  7. 激活完成！
